# Metagenomics Profiling Workshop

Source for the **QIB Core Bioinformatics** workshop website on metagenomics profiling.

The site covers taxonomic and functional profiling of metagenomic data using:

- **Kraken2** — k-mer-based taxonomic classification
- **MetaPhlAn** (Biobakery) — marker-gene-based taxonomic profiling
- **HUMAnN** (Biobakery) — functional profiling of microbial communities

## Website

The site renders at: http://corebio.info/metagenomics-profiling-workshop/
