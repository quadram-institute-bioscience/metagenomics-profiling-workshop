title: The Linux command line
---

We will be using the Linux command line to run all the tools in this course. 
The command line is a text interface to your computer, where you type commands to perform specific tasks. 
It is a powerful way to interact with your system, especially for bioinformatics workflows.

It can feel a bit intimidating at first, but with practice, it becomes a very efficient way to work with data.
In fact, the alternative might look like:

![A crazy GUI for fastp]({{ site.baseurl }}{% link img/fastp-gui.png %})

A terminal emulator is any program that allows you to access the command line interface (CLI) of your operating system.
For our training we will be using [CLIMB](https://www.climb.ac.uk/) which is a cloud-based platform that provides access to a Linux environment with pre-installed bioinformatics tools.

In other cases you might have something like:

![A terminal]({{ site.baseurl }}{% link img/terminal.png %})

We will start by familiarising with some commands but please don't worry if you don't understand everything at once. The goal is to get comfortable with the command line and learn how to navigate and manipulate files and directories.

We will always:

* Explain what each command does
* Provide full commands in this website (updated after the training!)
